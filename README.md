# Blog in React.js
A blog application to learn React.js

### Installation
0. Clone the repository:

		git clone https://github.com/marcelonaegeler/blog-react.git

0. Go to de folder:

		cd blog-react

0. Install Node.js dependencies:

		npm install

0. Install Bower dependencies:
	
		bower install

0. Run it:

		node app.js

0. Go to app:

		http://localhost:3000/
